## 默认mac快捷键
    command + d             跳转到定义
    command + shift + r     查找所有应用
    command + f             查找当前页面
    command + shift + f     查找整个项目
    command + /             注释相关
    command + [ or ]        缩进相关
    command + F10           运行到光标位置
    control + i             格式化页面
    control + command + 左  向后导航
    control + command + 右  向前导航
